## Principi guida della ricerca

1. **Evidence-based design**: ogni decisione e supportata da ricerca scientifica o dati di settore
2. **B2B-first**: il target primario e il professionista HoReCa con P.IVA
3. **Mobile-first**: il 78% del traffico e-commerce e mobile (Statista, 2024)
4. **Conversione**: ridurre il tasso di abbandono carrello dal 70% medio di settore (Baymard Institute)
5. **Accessibilita**: conformita WCAG 2.1 AA come requisito minimo

## Fonti principali
- Baymard Institute - 200.000+ ore di ricerca UX, 4.400+ sessioni di test
- Nielsen Norman Group - Ricerca UX dal 1998
- Statista - Dati di mercato e-commerce
- Google Web Vitals - Metriche performance
